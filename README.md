# 賽車可遊玩版 V6.6：加速手感強化版

這版依照最新需求調整：玩家車的抓地力再加強，左右轉彎更貼地、更不滑；AI 稍微減弱一點，但 6 車模式仍保留一台王牌電腦，讓比賽還是有壓力。

## 更新重點

- 抓地力再加強：車子左右轉向更穩，不會一直滑出去
- 側向慣性再降低：放開方向後更快回正
- 手機滑動轉向變得更細膩，較不容易一下打太滿
- 自動漂移門檻提高：普通過彎更偏抓地，不會一直誤觸漂移
- 車身 yaw / roll 再收斂：畫面更穩、更像貼地跑
- AI 稍微減弱：一般電腦車速度、超車慾望與切線能力小幅下降
- 6 車模式仍保留王牌 AI：還是會有一台比較強，但不會像前版那麼壓迫
- 保留速度上限：約 320 km/h
- 保留 3、2、1、GO 倒數起跑
- 保留撞牆減速：不扣血，但撞牆會掉速
- 保留 6 色車款、單人 / 3 台 / 6 台比賽、1 / 3 / 5 圈
- 保留追尾 / 近距兩種視角

## 操作方式

手機：
- 啟動：開始倒數 / 暫停
- 加速：長按加速
- 煞車：長按煞車
- 左右滑動：轉向
- 高速大角度滑動：自動漂移

電腦：
- Space：開始倒數 / 暫停
- W / ↑：加速
- S / ↓：煞車
- A / ←、D / →：轉向
- C：切換視角
- R：重置

## 執行方式

直接雙擊 `index.html` 即可遊玩，不需要安裝 npm 或開伺服器。
