# Neon Racer 3D V3

Three.js 霓虹 3D 賽車遊戲原型 V3。

## 新增內容

### 音效與音樂

- 引擎聲：速度越快，音高越高
- 氮氣聲：按住 Shift 時出現
- 漂移聲：高速轉向時出現
- 碰撞聲：撞牆、撞 AI 車時出現
- 背景音樂：電子低音循環

因為瀏覽器安全限制，音效必須在玩家按下「開始比賽」後才會啟動。

### 進階 AI 行為

AI 不再只是繞圈：

- 追擊：AI 落後且靠近玩家時，會嘗試追上玩家
- 阻擋：AI 領先且玩家靠近時，會往玩家前方卡位
- 超車：AI 靠近玩家後方時，會嘗試從側邊切線
- 巡航：距離玩家較遠時，沿賽道正常行駛

## 操作

- W / ↑：加速
- S / ↓：煞車
- A / ←：左轉
- D / →：右轉
- Shift：氮氣加速
- R：切換雨天
- T：切換日夜

## 開啟方式

直接打開 `index.html`。

若瀏覽器阻擋 ES module，請在資料夾內執行：

```bash
python3 -m http.server 8000
```

再打開：

```text
http://localhost:8000
```
