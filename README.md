# Racing Playable V6.9 — 手機橫式版面優化

這版以 V6.8 為基礎，重點修正手機瀏覽器橫式遊玩時的跑版問題。

## 更新重點

- 手機橫式版面重新整理
- HUD 在橫式手機固定成 6 欄，不會擠成直式排版
- 能量條縮小並置中，減少擋住賽道
- 啟動 / 煞車 / 加速按鈕重新配置，保留乾淨 UI
- 重置 / 視角 / 音效按鈕改成較小的橫式版面
- 設定畫面支援橫式小高度螢幕，可以滑動，不會超出畫面
- 直式手機會顯示「請轉成橫式」提示
- 加入 `100svh / 100dvh / visualViewport` 修正，降低 iPhone Safari 地址列造成的版面跳動
- 保留 V6.8 強化 AI、雙強敵、音樂音效、320 km/h、倒數起跑

## 使用方式

解壓縮後直接打開：

```txt
racing-playable-v6.9-mobile-landscape/index.html
```

手機建議橫式遊玩。若仍看到瀏覽器工具列擋住畫面，可以先點一下畫面或加入主畫面後再開啟，顯示空間會更穩。
