# Racing Playable V7.0 Mobile Fix

手機橫式版面修正版。

## 更新重點

- 直式手機不再顯示巨大設定面板，只顯示「請轉橫式」提示。
- 橫式手機設定面板改成更扁、更寬，減少 Safari 地址列造成的跑版。
- 車色改成橫向 6 欄，在橫式畫面更省空間。
- 保留 V6.8 / V6.9 的強化 AI、音效、320 km/h、倒數起跑、6 車對戰。

## 使用方式

解壓縮後直接開啟 `index.html`。手機請橫式遊玩。

若部署到 GitHub Pages，請把 `index.html`、`style.css`、`js/game3d.js` 一起覆蓋。
